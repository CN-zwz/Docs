## 2025年4月30日

## 更改页脚，增加访问域名提示
本次更新启用了 Footer 组件，增加了访问域名提示。
原理就是在页脚插入一个 `<div>`，然后通过 JavaScript 判断当前访问的域名，来显示不同的提示信息。

具体提示见 ```FooterDomainText.js```：[github](https://github.com/CN-zwz/Docs/blob/master/src/components/FooterDomainText.js)或[jsdelivr](https://cdn.jsdelivr.net/gh/CN-zwz/Docs@master/src/components/FooterDomainText.js)。
